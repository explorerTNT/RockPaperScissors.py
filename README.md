# Камень, Ножницы, Бумага (Rock, Paper, Scissors)

Учебный пет-проект — реализация классической игры "Камень, Ножницы, Бумага" на языке Python.

## Описание проекта

Этот проект представляет собой консольную версию популярной игры "Камень, Ножницы, Бумага", разработанную на Python в учебных целях. Проект демонстрирует базовые концепции программирования, такие как условные операторы, случайный выбор, обработка пользовательского ввода и организация кода.

## Особенности

- Простой и интуитивно понятный интерфейс командной строки
- Случайный выбор хода компьютера
- Валидация пользовательского ввода

## Требования

- Python 3.6 или выше
